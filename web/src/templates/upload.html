How It Works
Upload Process:
  1. File saved to /tmp/uploads/
  2. ClamAV scans for malware
  3. Calculate SHA256 hash
  4. If CLEAN:
    • Move to /uploads/safe/
    • Create DB record
    • Return success
  5. If INFECTED:
    • Move to /uploads/quarantine/
    • Send alert to VM1
    • VM1 blocks uploader IP
    • Return error
Architecture

Critical: VM2 does NOT see real client IPs!

  • VM2 only sees: 10.0.0.1 (VM1)
  • Alert sent to VM1: filename, hash, signature
  • VM1 correlates with conntrack
  • VM1 blocks real attacker IP
